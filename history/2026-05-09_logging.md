# 2026-05-09 — Logging System

## New Files

### `src/common/logging.hpp` / `logging.cpp`
Project-wide logging module written from scratch.

- Compile-time level (`#if LOG_LEVEL <= ...`) — lower-level code fully stripped from release builds
- Runtime level (`logger::g_runtime_level`) — dynamic filtering at runtime
- Levels: TRACE(0) / DEBUG(1) / INFO(2) / WARN(3) / ERROR(4) / NONE(5)
- ANSI color output — automatically disabled on `NDEBUG` builds
- CSV log file — created automatically based on run timestamp when `logger::set_log_file("logs")` is called
  - Format: `logs/YYYY-MM-DD_HHMMSS.csv`
  - Columns: `timestamp, level, file, line, func, message`
- Macros: `LOG_TRACE`, `LOG_DEBUG`, `LOG_INFO`, `LOG_WARN`, `LOG_ERROR`
- `__FILE__`, `__LINE__`, `__func__` injected automatically
